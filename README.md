# Secure Login System

## Features

- User Registration
- Secure Login
- Password Hashing (bcrypt)
- SQLite Database
- SQL Injection Protection
- Input Validation
- Session Management
- Logout
- Google Authenticator 2FA
- Flask-WTF CSRF Protection

## Technologies

- Python
- Flask
- SQLAlchemy
- Flask-Bcrypt
- Flask-Login
- Bootstrap
- SQLite
- PyOTP

## Installation

pip install -r requirements.txt

python app.py
